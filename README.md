# Lib_Book_Management_System
C Project for managing library books.
Overview: This project is a console-based application created to aid librarians to manage issued book records and organize books efficiently. 
Features:
1. Add book
2. Delete book
3. Display list of books
4. Issue book
5. Display list of issued books
6. Cross platform screen clearing (cls for Windows, clear for Linux/Unix).
   Prerequisites
1. GCC or any C compiler
2. A terminal/command prompt
